Level Generator

Download the unity package: HERE

This was a tool i made to make level creation simpler for several top down projects that i was working on. This tool allows you to place prefabs in a grid based level based on a given sprite.

To achieve this what i did was have the script scan a given image pixel by pixel and place a prefab at the pixels coordinates in the game world based on the colour of the pixel and a user defined prefab.

Prev
Next

One limitation of this system is that prefabs must be the same size otherwise they may overlap.

Another limitation is that the colours for each prefab have to be defined by the player. This means that if the players rgb values are slightly different from the colour values on the image, the world generation will fail.
