This is a tool that iâ€™ve been making for unity for about a week now. My goal with this project was to make a system for creating Json files which would make it easier for me to build rpg games later on.

To do this i decided to create a custom editor script which would give me a node based canvas which you can pan around in making it easy to visualise what each character says in response to what reply the player selects.

Prev
Next

Each window in the editor has a speech section where the npc speech goes, below that there is the option to add replies, each reply consists of a text area, and a little sun icon which allows you to connect it to another speech it additionally has optional requirements and rewards.

For my first proper custom editor window I am very happy with the results, the only things which I have no implamented is a right click function which would bring up a dialogue to add new speeches, and save/load functionality from within the custom editor.
