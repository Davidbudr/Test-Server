Willows Cookbook

Willows Cookbook can be downloaded HERE.

This is my second mod for Don’t Starve Together, this mod allows players to craft unique character items which would normally only be accessible to certain characters. I made this mod with small and casual personal servers in mind, as it is a mod which allows just a few players to use what would normally be reserved exclusively for larger groups.

Prev
Next

The way in which this mod works is two fold.
Initially, character items such as “Willows Lighter” or “Portable Crockpot”, are given new recipies in a custom tab. This is the primary goal of the mod.

However, the mod has one other utility. with this mod, characters who would otherwise be unable to use certain items are given that functionality through that games use of “Tags”, this had the unintended side effect of giving characters other abilities which were also attached to these character tags i.e. faster woodcutting.

Making this mod gave me an idea for a more balanced mod which is still in its infancy where players would be able to craft equipable items which give them the powers of other characters, this would be achieved by having the item add a character tag to the player when equiped and remove said tag when it is unequiped.
