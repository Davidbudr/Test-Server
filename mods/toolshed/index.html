Toolshed

Toolshed can be downloaded HERE.

This was my first attempt at a mod for the popular multiplayer game ” Don’t Starve Together ”, It was also my first attempt at Lua. The goal of this mod is to allow players to change the durability of Golden tools, an upgraded version of regular tools provided in game.

This mod does change the game quite a lot by rebalancing those tools, however the choices of how much it rebalances these tools is completely up to the player. Giving options from half durability to infinite durability. This was my first entry into lua as a programming language.
